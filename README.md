# Light

Light is a patient-first clinical trial matching and community platform demo.

It helps patients and caregivers organize health context, find potentially relevant clinical trials, understand what needs confirmation, learn from synthetic community examples, save trials, and prepare outreach for a care team discussion.

Light is not medical advice and does not determine trial eligibility.

## Run Locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Validate

```bash
npm run typecheck
npm run build
```

## Demo Path

1. Select a demo case on the home page.
2. Review the extracted profile.
3. Find potentially relevant trials.
4. Save trials and view details.
5. Review synthetic community context.
6. Generate outreach prep.

## Optional Environment

External adapters are optional. The demo works without credentials.

```bash
NIA_API_KEY=
HYPERSPELL_API_KEY=
```
